Sobre Cerberus

Cerberus Gestor de Contraseñas es una aplicación de escritorio y móvil diseñada bajo la filosofía local-first (offline-first). Esto significa que todas tus contraseñas, datos sensibles y configuraciones se almacenan exclusivamente en tu dispositivo, bajo tu control total.

No existe publicidad, no hay rastreadores, no hay conexión forzada a servidores remotos. La privacidad y seguridad no son características opcionales, sino el fundamento de la arquitectura de Cerberus.

🔒 Principio Core: Tu información sensible nunca abandona tu dispositivo a menos que TÚ decidas explícitamente sincronizarla con tus propios servicios en la nube (Google Drive personal).

Arquitectura Criptográfica

Cerberus implementa una arquitectura de seguridad multicapa que garantiza la protección máxima de tus datos sensibles. Cada componente está diseñado con rigurosos estándares criptográficos reconocidos internacionalmente.

📊 Especificaciones Técnicas Detalladas

1. Base de Datos Local Blindada (SQLCipher)

La base de datos interna de Cerberus está completamente cifrada mediante SQLCipher, un motor de base de datos SQL que integra encriptación transparente a nivel de página. Especificaciones:

  • Algoritmo de Cifrado: AES-256 en modo CBC (Cipher Block Chaining)
  • Función de Derivación de Clave: PBKDF2-HMAC-SHA256
  • Iteraciones de Derivación: 600.000 iteraciones (estándar NIST actualizado)
  • Tamaño de Página: 4096 bytes encriptados
  • Validación de Integridad: HMAC para detectar alteraciones
2. Copias de Seguridad Manuales Encriptadas (.cjson)

Exporta tus datos en formato JSON encriptado (.cjson) con tecnología de vanguardia para máxima durabilidad y portabilidad. Especificaciones:

  • Algoritmo de Derivación de Clave: Argon2id (resistente a ataques GPU/ASIC)
  • Parámetros Argon2id: Configurados según recomendaciones OWASP 2023
  • Algoritmo de Cifrado: AES-256-GCM (Galois/Counter Mode)
  • Autenticación: Tag de autenticación integrado (GCM authentication tag)
  • Formato de Salida: JSON estructurado con metadatos criptográficos
  • Portabilidad: Importable directamente en otra instancia de Cerberus
3. Sincronización y Backup en la Nube Privada (Google Drive)

La sincronización con Google Drive es completamente opcional y está diseñada para máxima privacidad. Especificaciones:

  • Interacción API: Exclusivamente con la carpeta restringida "appDataFolder"
  • Cifrado End-to-End: Los datos se cifran ANTES de salir de tu dispositivo
  • Aislamiento de Datos: Google no puede acceder a los datos descifrados
  • Sincronización Selectiva: Controla qué datos se sincronizan y cuándo
  • Revocación de Permisos: Desactiva la sincronización en cualquier momento
  • Cumplimiento: Sigue políticas de privacidad de Play Store y Google Cloud
⚠️ Nota de Seguridad Crítica: En el caso improbable de que alguien acceda a tu dispositivo o a los archivos de respaldo, los datos permanecen 100% protegidos detrás de múltiples capas de criptografía. La única forma de acceder es con tu contraseña maestra.
